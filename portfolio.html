Nexora

ReactNode.jsSaaS

The Problem

Logistics companies needed a unified dashboard to track fleets, manage routes, and analyze delivery performance in real-time.

Our Solution

We built Nexora — a real-time SaaS dashboard with live map tracking, route optimization, delivery analytics, and team management. Built with React for a fluid UI and Node.js for real-time WebSocket updates.

Tech Stack

React, Node.js, Express, Socket.io, PostgreSQL, Mapbox GL, Chart.js, Docker, AWS

Outcome

30% reduction in delivery times for pilot clients. 99.9% uptime since launch. Onboarded 12 logistics companies within the first quarter.

Bloom

FlutterFirebase

The Problem

Young professionals needed an accessible, beautifully designed app for daily mental wellness practices — meditation, journaling, and mood tracking.

Our Solution

Bloom is a cross-platform mobile app built with Flutter and Firebase. Features include guided meditation sessions, a journaling tool with prompts, mood tracking with weekly insights, and push notification reminders.

Tech Stack

Flutter, Dart, Firebase (Auth, Firestore, Cloud Functions), RevenueCat, Lottie Animations

Outcome

10,000+ downloads in the first month. 4.8★ average rating. Featured in "Apps We Love" on Product Hunt.
